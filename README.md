### Run locally
**Option 1 (Node):**
```bash
npm start
```**Option 2 (Python, if npm start fails):**
```bash
npm run run
```
Then open **http://localhost:3000** in your browser.

You can also open `index.html` directly in your browser.

### Enabling GitHub Pages (one-time)
1. Open the repo on GitHub → **Settings** → **Pages**.
2. Under **Build and deployment**, set **Source** to **GitHub Actions**.
3. Every push to `main` will deploy the site. The live URL is `https://vshah513.github.io`. Main portfolio lives at [portfolio-vshah.vercel.app](https://portfolio-vshah.vercel.app) (source: [github.com/Vshah513/Portfolio](https://github.com/Vshah513/Portfolio)).
